# smart-baby-monitor
The Smart Baby Monitor is an IoT-based embedded system designed to enhance infant safety by continuously monitoring environmental conditions and detecting unusual activity such as crying or movement. This system uses an ESP32 microcontroller to collect data from multiple sensors including a sound sensor, motion sensor (PIR), and a temperature etc. 
# 🍼 Smart Baby Monitor

## 📌 Overview
This project is an IoT-based baby monitoring system using ESP32.

## ⚙️ Features
- Cry detection
- Motion detection
- Temperature monitoring
- Alerts via WiFi

## 🔧 Components
- ESP32
- DHT11 Sensor
- Sound Sensor
- PIR Sensor
